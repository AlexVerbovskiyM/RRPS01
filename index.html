Ця лабораторна робота має на меті ознайомлення студентів з основами роботи з системою контролю версій Git.

Git є потужним інструментом для управління змінами в коді та полегшує співпрацю між розробниками.

Використання Git допомагає зберігати історію змін, працювати в командах і уникати конфліктів при внесенні змін у код.
